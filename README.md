# Camper-rental

A personal project that helps to implement the most daring ideas and choose a camper for rent without any obstacles.


## Tech Stack

**Client:** React, Redux

**Server:** mockapi.io


## Features

- Authentication
- Adding to favorites
- Search by location
- Filter by properties
- Online booking
